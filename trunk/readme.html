CR_CodeTweet

Description

Sometimes you have a snippet of code in Visual Studio that you'd like to share over Twitter. You used to have to post the code snippet somewhere (or take a screen shot), then manually tweet the link to that snippet.

Not anymore!

CR_CodeTweet is a plugin for DXCore that adds a context menu and hotkey that allows you to select code in Visual Studio and automatically post it to CodePaste.NET, then tweet a link to that posted snippet - all without leaving Visual Studio.

Requirements

This product requires DXCore for Visual Studio .NET 9.1.3.0 or later.

DXCore is a free download from Developer Express, Inc.: http://www.devexpress.com/Products/Visual_Studio_Add-in/DXCore/

Installation

  1. Make sure all instances of Visual Studio are closed.
  2. Copy the CR_CodeTweet.dll assembly into your DXCore plug-ins folder. This is typically something like:
    C:\Documents and Settings\YOURUSERNAME\My Documents\DevExpress\IDE Tools\Community\PlugIns
    OR
    C:\Program Files\DevExpress 2009.1\IDETools\Community\PlugIns
  3. Start Visual Studio.
  4. From the DevExpress menu, select "Options."
  5. Expand the "Editor" folder in the list of options and select the "CodeTweet" option panel.
  6. Click the "Authorize CodeTweet" button to run through the process of enabling the plugin to tweet on your behalf.
  7. Fill in your username/password credentials for CodePaste.NET. (Unfortunately the CodePaste.NET web service does not support OpenID, so if you currently use OpenID on that site you will need to switch to username/password.)
  8. Click "Apply" to save your options.
  9. Optional: Bind a hotkey to the tweet functionality.
    1. Expand the "IDE" folder in the list of options and sleect the "Shortcuts" option panel.
    2. Add a new shortcut key and bind it to the "Tweet Selected Code" command. Something easy to remember like "Ctrl+Shift+Alt+T" (for "Tweet") is recommended.
  10. Click OK to save your options and close the dialog.

Usage

  1. Open an instance of Visual Studio.
  2. When working in a source file, select one or more characters and tweet the snippet by either:
    • Right-clicking and selecting "Tweet This!" from the context menu, OR
    • Using the hotkey you set up to tweet the selected code.
